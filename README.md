# **Compass Marketing Analytics Africa**

**Compass Marketing Analytics Africa** is a modern web-based platform designed to help organizations across Africa easily create surveys, collect feedback across multiple channels, and transform responses into powerful real-time insights.

Built with speed, scalability, and ease of use in mind, Compass enables businesses, NGOs, research teams, and agencies to better understand their audience and make informed decisions backed by data.

---

## **Key Features**

### **Survey Builder**

* Simple, intuitive interface
* Multiple question types
* Custom themes & branding
* Works across all devices

### **Multi-Channel Distribution**

* Share via links, QR codes, email, or SMS
* Collect responses online or offline
* Built for regions with unstable connectivity

### **Real-Time Analytics**

* Live dashboards
* Charts & visualizations
* Segmentation and filtering
* Export to CSV/Excel

### **Secure & Reliable**

* Optimized for performance
* Clean, modern UI with Tailwind + React
* Scalable architecture

---

## **Technology Stack**

* **React (Vite + SWC)**
* **TypeScript**
* **Tailwind CSS**
* **shadcn/ui**
* **Vite** for fast development
* Designed for deployment on **Vercel**, **Netlify**, or cloud environments

Compass empowers African teams to run research, gather community insights, evaluate programs, measure customer satisfaction, and make smarter data-driven decisions—without needing advanced technical skills.
